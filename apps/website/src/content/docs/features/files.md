# File Management

## Overview
Asoode provides a complete file management system for your team. Upload, organize, and share files with a familiar folder-based interface.

## Features
- **Personal storage**: Each user has their own file space
- **Shared folders**: Share files with team members
- **Folder organization**: Create nested folder structures
- **File preview**: View documents, images, and PDFs inline
- **Task attachments**: Attach files directly to tasks
- **Storage quotas**: Manage storage per group

## Uploading Files
1. Navigate to the Files section
2. Click "Upload" or drag and drop files
3. Choose the destination folder
4. Files are immediately available to authorized members

## Sharing Files
- Share individual files or entire folders
- Set permissions (view, edit, manage)
- Generate shareable links
- Track who has accessed shared files

## Storage Management
- View storage usage per group
- Monitor per-member usage
- Upgrade plans for more storage
- Clean up unused files regularly

## Supported File Types
- Documents: PDF, DOC, DOCX, XLS, XLSX, PPT, PPTX
- Images: PNG, JPG, GIF, SVG, WebP
- Archives: ZIP, RAR
- Code: TXT, JSON, XML, CSV
