# Projects

## Overview
Projects are the main containers for organizing work in Asoode. They hold work packages, members, and settings that define how your team collaborates.

## Project Types
- **Simple**: A single-level project ideal for straightforward tasks
- **Complex**: Supports sub-projects and seasons for larger initiatives

## Creating a Project
1. Navigate to your group
2. Click the "+" button or "New Project"
3. Choose a project type (Simple or Complex)
4. Enter a name and optional description
5. Set privacy and access levels

## Project Features
- **Sub-projects**: Break large projects into smaller, manageable units
- **Seasons**: Define time-based phases for your project
- **Milestones**: Track key project checkpoints
- **Road Map**: Visualize project timeline
- **Tree View**: See the complete project hierarchy
- **Progress Tracking**: Monitor completion with charts

## Managing Members
Each project can have its own set of members with specific access levels. Members can be added from the parent group or invited directly.

## Tips
- Use Complex projects for ongoing work with multiple phases
- Set milestones early to keep your team focused
- Review the tree view regularly to understand project scope
