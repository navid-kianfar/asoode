# Messenger

## Overview
Asoode's built-in messenger provides real-time communication for your team, right where the work happens. No need to switch between separate tools.

## Features
- **Direct messages**: One-on-one conversations
- **Group channels**: Project and team conversations
- **File sharing**: Share files within conversations
- **Real-time delivery**: Powered by WebSocket
- **Threading**: Reply to specific messages
- **File previews**: View images and documents inline

## Getting Started
1. Click the Messenger icon in the sidebar
2. Start a new conversation or select an existing one
3. Type your message and press Enter to send

## Channel Types
- **Direct**: Private conversation between two people
- **Project channel**: Automatically created for each project
- **Group channel**: Conversations tied to organizational groups

## File Sharing
- Drag and drop files into the conversation
- Click the attachment icon to browse files
- Supported previews: images, PDFs, documents

## Tips
- Use project channels for work-related discussions
- Direct messages for quick, private conversations
- Pin important messages for easy reference
- Use @mentions to notify specific team members
