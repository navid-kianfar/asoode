# Tasks

## Overview
Tasks are the individual items of work in Asoode. Each task is a rich workspace where team members can collaborate, track progress, and communicate.

## Task Properties
- **Title**: Brief description of the work
- **Description**: Rich text with formatting and mentions
- **Members**: Assigned team members
- **Labels**: Color-coded categories
- **Due date**: Deadline with reminders
- **State**: Current status (e.g., open, in progress, done, blocked)
- **Priority**: Task importance level

## Advanced Features
- **Sub-tasks**: Break a task into smaller items
- **Time tracking**: Log time spent on the task
- **File attachments**: Attach documents and files
- **Comments**: Discuss with threading and privacy options
- **Activity log**: See every change made
- **Voting/Polls**: Team decision-making
- **Watch**: Get notified of changes

## Task States
Tasks can be moved through customizable states:
1. Open
2. In Progress
3. Done
4. Blocked
5. Archived

## Tips
- Write clear, actionable task titles
- Break large tasks into sub-tasks
- Use comments instead of creating new tasks for discussions
- Track time as you work for accurate reports
